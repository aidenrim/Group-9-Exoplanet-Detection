Project Overview This document formally initiates the Exoplanet Detection project undertaken by Group 9 The project aims to build a machine learning pipeline capable of detecting exoplanets from stellar photometric data and predicting key planetary characteristics. The system will be exposed through an interactive web application An exoplanet is a planet that orbits a star outside our solar system One of the most reliable detection techniques is the Transit Method: when an exoplanet passes in front of its host star, it causes a measurable, periodic dip in the star's observed brightness. This project exploits that phenomenon through deep learning models trained on data from NASA's Kepler and TESS missions 2. Methods & Al Techniques 5.1 Data Pre-processing Retrieval of stellar flux time series (light curves) using the Astroquery MAST API. Normalisation and detrending Of light curves to remove systematic noise and stellar variability Image and signal processing using OpenCV for feature extraction and noise reduction Phase-folding Of time series based on candidate transit periods to produce folded light curve representations 5.2 Detection Model Primary architecture: Convolutional Neural Network (CNN) operating on ID or 2D representations of light curves. Investigation Of ResNet-18 as a transfer-learning baseline for image-based light curve representations Binary classification output: exoplanet candidate (positive) vs. non-exoplanet (negative) Model will produce a calibrated confidence score for each prediction 5.3 Prediction Models Exoplanet radius estimation using transit depth relative to stellar radius (regression model). Orbital period prediction using detected transit timing interva s (regression model or signal periodicity analysis) Potential use of Deep Q-Networks or additional neural network architectures for comparative evaluation. 5.4 Web Application Streamlit-based interactive web application to visualise light curves, detection outputs, and planetary predictions.
